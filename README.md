# UTHKARSHA

AyuRaksha LifeVault - A comprehensive health management platform.

## Tech Stack

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, Prisma

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install
```

### Running the Application

```bash
# Start both servers (Windows)
start.bat
```

Or run individually:

```bash
# Backend (port 5000)
cd backend && npm start

# Frontend (port 3000)
cd frontend && npm run dev
```
